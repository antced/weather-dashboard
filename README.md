# Weather Dashboard

![screenshot](./assets/images/screenshot.png)

## Description

https://antced.github.io/weather-dashboard/

Current and 5-day weather dashboard that uses OpenWeatherMap APIs for weather data. Includes the following features:

- Bootstrap
- jQuery
- Local Storage
- Clearable History
